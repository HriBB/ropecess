## Installation

```bash
# install dependencies
pnpm install

# copy .env file
cp .env.example .env

# run development server
pnpm dev
```

## Deployment

Deployed to [Netlify](https://www.netlify.com/) (see `netlify.toml` and [ADR-0002](docs/adr/0002-deploy-to-netlify.md)).
